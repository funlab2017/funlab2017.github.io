致申请学生

我们乐于接纳自励并且在数学、网络和通信理论方面基础扎实的学生。如果您有意研究优化理论、 信号处理、机器学习和博弈论在信息工程专业中的应用，并且对我们某个研究方向感兴趣， 烦请您通过邮件与课题相关的教授取得联系，并附上您的简历，另外请将附件发送给付立群教授。 目前可提供的研究岗位包括：

  • 全日制的博士后研究员 ( 付立群教授 )
  • 全日制博士 ( 付立群教授 )
  • 全日制硕士研究生 ( 付立群教授、 王德清教授、 吕江滨教授、游理钊教授)
  • 访问学者及访问博士生( 付立群教授)
